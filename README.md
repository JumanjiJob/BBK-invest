# BBKinvest ИИ-консультант

Интеллектуальный ассистент для сайта BBKinvest с функцией сбора заявок и отправкой в Telegram.

## Основные функции
- Определение типа пользователя (физлицо, бизнес, инвестор)
- Консультация по соответствующим продуктам
- Сбор заявок в унифицированном формате
- Отправка заявок в Telegram-бота
- Админ-панель для управления контентом

## Структура проекта
- `/backend` - серверная часть (API, логика диалога)
- `/frontend` - клиентская часть (чат-виджет, админка)
- `/admin` - админ-панель (веб-интерфейс)
- `/docs` - документация
- `/database` - файлы базы данных
- `/tests` - тесты
- `/scripts` - вспомогательные скрипты

## Технологии
- Бэкенд: Python + FastAPI/Flask
- Фронтенд: JavaScript, HTML, CSS
- База данных: PostgreSQL
- Интеграции: Telegram Bot API

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Настроить переменные окружения (см. .env.example)
4. Запустить: `python main.py`
