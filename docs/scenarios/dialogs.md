### 1. ОБЩАЯ ЛОГИКА ДИАЛОГА

```
ПРИВЕТСТВИЕ:
  ИИ: "Здравствуйте! Я ИИ-консультант BBKinvest. Чем могу помочь?"

ПЕРВЫЙ ВОПРОС (определение пути):
  ИИ: "Вы рассматриваете получение займа или хотите инвестировать?"
  Варианты ответа (кнопки): ["Займ", "Инвестировать"]
```

### 2. СЦЕНАРИЙ 1: ФИЗИЧЕСКОЕ ЛИЦО (ЗАЕМЩИК)

```
ЕСЛИ ответ = "Займ":
  ВТОРОЙ ВОПРОС:
    ИИ: "Займ оформляется на физическое лицо или на бизнес (ЮЛ/ИП)?"
    Варианты: ["Физическое лицо", "Бизнес"]
  
  ЕСЛИ ответ = "Физическое лицо":
    АКТИВАЦИЯ СЦЕНАРИЯ:
      ИИ: "Консультирую по займам под залог автомобиля или недвижимости. Для оформления заявки потребуется несколько данных."
    
    ЗАПРОС ДАННЫХ (последовательно):
      1. ИИ: "Введите ваше имя:"
         Сохранить: user_data.name
      
      2. ИИ: "Укажите залог: марку, модель и год выпуска авто или описание недвижимости:"
         Сохранить: user_data.collateral
         Примечание: Можно добавить уточнение "Например: Kia Sportage, 2021 год"
      
      3. ИИ: "Желаемая сумма займа (в рублях):"
         Валидация: число, > 0
         Сохранить: user_data.amount
      
      4. ИИ: "Цель займа:"
         Сохранить: user_data.purpose
      
      5. ИИ: "Введите номер телефона для связи:"
         Валидация: российский номер (10-11 цифр)
         Сохранить: user_data.phone
    
    ПОДТВЕРЖДЕНИЕ:
      ИИ: "Спасибо! Проверьте данные:\n\nИмя: {name}\nЗалог: {collateral}\nСумма: {amount} руб\nЦель: {purpose}\nТелефон: {phone}\n\nВсё верно?"
      Варианты: ["Да, отправить заявку", "Нет, исправить"]
    
    ЕСЛИ подтверждено:
      Отправка в Telegram по формату
      ИИ: "Заявка отправлена! Специалист свяжется с вами в ближайшее время. Спасибо!"
      Конец диалога.
    
    ЕСЛИ НЕ подтверждено:
      Возврат к редактированию (предложить выбрать поле для исправления)
```

### 3. СЦЕНАРИЙ 2: БИЗНЕС (ЮЛ/ИП, ЗАЕМЩИК)

```
ЕСЛИ ответ = "Займ" → "Бизнес":
  АКТИВАЦИЯ СЦЕНАРИЯ:
    ИИ: "Консультирую по займам для юридических лиц и ИП. Уточните, какое обеспечение имеется:"
    Варианты обеспечения (кнопки/текст): 
      ["Недвижимость (офис, склад, производство)", 
       "Движимое имущество (оборудование, транспорт, техника)", 
       "Интеллектуальная собственность (патенты, товарные знаки)"]
    Примечание: Можно выбрать несколько вариантов
    
  ЗАПРОС ДАННЫХ:
    1. ИИ: "Укажите полное название компании или ФИО с указанием 'ИП':"
       Примеры: 
         - Для ООО: "ООО 'ТехноПром'"
         - Для ИП: "ИП Иванов Игорь"
       Сохранить: user_data.company_name
    
    2. ИИ: "Желаемая сумма займа (в рублях):"
       Валидация: число, > 0
       Сохранить: user_data.amount
    
    3. ИИ: "Опишите обеспечение подробно (можно несколько видов):"
       Подсказка: "Например: Станки (оборудование 2023 г.), товарный знак 'Марка'"
       Сохранить: user_data.collateral
    
    4. ИИ: "Цель займа:"
       Сохранить: user_data.purpose
    
    5. ИИ: "Контактный телефон для связи:"
       Валидация: российский номер
       Сохранить: user_data.phone
  
  ПОДТВЕРЖДЕНИЕ:
    ИИ: "Проверьте данные:\n\nКомпания: {company_name}\nСумма: {amount} руб\nОбеспечение: {collateral}\nЦель: {purpose}\nТелефон: {phone}\n\nВсё верно?"
    Варианты: ["Да, отправить", "Нет, исправить"]
  
  Отправка/редактирование аналогично сценарию 1
```

### 4. СЦЕНАРИЙ 3: ИНВЕСТОР

```
ЕСЛИ ответ на ПЕРВЫЙ ВОПРОС = "Инвестировать":
  АКТИВАЦИЯ СЦЕНАРИЯ:
    ИИ: "Консультирую по инвестиционным продуктам под обеспечение залогового имущества. Для подбора варианта потребуется информация."
  
  ЗАПРОС ДАННЫХ:
    1. ИИ: "Введите ваше имя:"
       Сохранить: user_data.name
    
    2. ИИ: "Сумма для инвестирования (в рублях):"
       Валидация: число, > 0
       Сохранить: user_data.investment_amount
    
    3. ИИ: "Горизонт инвестирования (в месяцах):"
       Валидация: число, 1-120
       Сохранить: user_data.term_months
    
    4. ИИ: "Цель инвестирования:"
       Варианты (кнопки): ["Пассивный доход", "Сохранение капитала", "Диверсификация", "Другое"]
       ЕСЛИ "Другое": запросить текстовый ввод
       Сохранить: user_data.investment_goal
    
    5. ИИ: "Контактный телефон для связи:"
       Валидация: российский номер
       Сохранить: user_data.phone
  
  ПОДТВЕРЖДЕНИЕ:
    ИИ: "Проверьте данные:\n\nИмя: {name}\nСумма: {investment_amount} руб\nСрок: {term_months} мес\nЦель: {investment_goal}\nТелефон: {phone}\n\nВсё верно?"
    Варианты: ["Да, отправить", "Нет, исправить"]
  
  Отправка/редактирование аналогично
```

### 6. ОБРАБОТКА ОШИБОК И ВАЛИДАЦИЯ

```
ДЛЯ КАЖДОГО ПОЛЯ ВВОДА:
  ЕСЛИ ввод не прошел валидацию:
    ИИ: "Пожалуйста, проверьте формат. [конкретное требование]"
    Повторный запрос поля (макс. 2 попытки, затем переход к менеджеру)

ЕСЛИ пользователь хочет начать заново:
  ИИ: "Начать диалог заново?" → сброс к ПРИВЕТСТВИЮ

ТАЙМАУТ: 15 минут неактивности → "Диалог завершен из-за неактивности. Для новой заявки перезагрузите виджет."
